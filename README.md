# o9ds Design System Website

A design system documentation site for **o9 Design Lab** — built from the o9 brand identity and [o9ds.figma.site](https://o9ds.figma.site/) inspiration.

## Features

- **Colors** — Brand palette (black/white) + product-specific colors (Shock, Indigo, Grass, Sun, Juice, Scarlet)
- **Typography** — Type scale and :o9 Sans-inspired font stack (DM Sans)
- **Components** — Buttons, inputs, cards, badges, alerts
- **Spacing** — 4px-based spacing scale
- **Icons** — Geometric icon set and product-specific usage

## Tech Stack

- React 18 + Vite
- Tailwind CSS
- React Router

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
```

Output in `dist/`.
